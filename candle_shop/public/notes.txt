## How to run frontend locally (quick)
1. `cd frontend`
2. `npm install`
3. Create `.env` from `.env.example` and set `VITE_API_URL` to your backend 
(e.g. `http://localhost:8080/api`).
4. `npm run dev` and open `http://localhost:5173`
